# Welcome to My Blog

Welcome to this minimal, markdown-based blog! This is a demonstration of how simple and elegant a blog can be when built with just HTML, CSS, and markdown.

## What Makes This Special?

This blog system is designed with simplicity at its core:

- **Pure Markdown**: All posts are written in simple markdown files
- **No Build Process**: No complex build tools or frameworks needed
- **Accessible**: Clean, semantic HTML that works everywhere
- **Fast**: Minimal JavaScript, maximum performance
- **Beautiful**: Clean typography and thoughtful spacing

## How It Works

1. Write your post in markdown
2. Add it to the `posts/` directory
3. Update the `posts.json` index
4. That's it!

The blog automatically renders your markdown into beautiful, readable HTML using a lightweight parser.
